## How To Use It
- Create your main function to test
- Everything is compiled this way:
  
  ```bash
  gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o printf
  ```
- Run by using:
  
  ```bash
  ./printf
  ```
